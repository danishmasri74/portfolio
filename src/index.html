Typing Speed Test

Typing Speed Test is a web application that measures your typing skills in words per minute (wpm), characters per minute (cpm) and typo errors. You have to type a paragraph that is given by the application. The application will start a 1-minute countdown timer when you begin typing and display your results when the time is up. You can see how many errors you made and how fast you typed. Typing Speed Test is a basic and helpful way to improve your typing speed and accuracy.

SimpleTetris

Simple Tetris is a browser-based game with a clean user interface design and basic features. This project is built using HTML, CSS, and JavaScript. It includes a highest score display panel, allowing players to track their progress and compete against others. The game follows the classic Tetris gameplay mechanics, where players aim to arrange falling blocks in a way that completes horizontal lines. As lines are cleared, the player's score increases.
